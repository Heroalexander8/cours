## ✨ Características del Proyecto

El sitio web cuenta con un diseño moderno tipo *Dashboard* e incluye las siguientes funcionalidades:

*   🖥️ **Interfaz UI/UX Responsiva:** Diseño limpio y adaptable, construido con HTML y CSS puro, utilizando una paleta de colores tecnológica y tipografía moderna (Inter).
*   📑 **Filtros Interactivos:** Sistema de pestañas programado con JavaScript para filtrar la vista de los cursos entre "Todos", "6to Semestre" y "7mo Semestre".
*   🎓 **Tarjetas de Cursos (Cards):** Cada materia cuenta con una tarjeta informativa que muestra el nombre del curso, el docente a cargo y una sumilla/resumen extraída del documento oficial.
*   📥 **Descarga Directa:** Botones integrados para visualizar y descargar los archivos PDF originales de cada sílabo alojados en el repositorio.
